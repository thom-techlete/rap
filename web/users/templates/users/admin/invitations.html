Hoe werken uitnodigingscodes?

Voor nieuwe spelers:

Nieuwe spelers hebben een uitnodigingscode nodig om zich te kunnen registreren. Zonder een geldige code kunnen zij geen account aanmaken.

Beheer opties:
  • Stel een maximaal aantal gebruiken in
  • Voeg een vervaldatum toe
  • Activeer/deactiveer codes wanneer nodig
  • Kopieer codes om te delen
